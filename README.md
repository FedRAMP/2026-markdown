# FedRAMP 2026 Markdown Content

This repository contains pure Markdown content for the FedRAMP Consolidated Rules for 2026.

The content is generated from two upstream sources:

- Structured rule data from `fedramp-consolidated-rules.json` in the FedRAMP Rules repository.
- Hand-written narrative content from the FedRAMP 2026 repository.

Together, these files are the underlying Markdown corpus that is rendered into HTML using Zensical.

The purpose of this repository is to expose the Markdown directly to AI agents and other text-oriented tools so they can work from the source content without crawling the rendered HTML site.

Source commit metadata is recorded in `_sources.json`.
