# Important Dates for the Consolidated Rules for 2026

!!! warning "Some of these dates are placeholders during Public Preview!"

| Milestone                                    | Date       | Explanation                                                                                                                                                                                   |
| :------------------------------------------- | :--------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Finalization of the Consolidated Rules for 2026       | 2026-06-30 | The Consolidated Rules for 2026 will be moved from Public Preview into final status no later than the end of FY26 Q3.                                                                         |
| Optional Early Adoption                      | 2026-07-01 | All stakeholders should begin incrementally adopting or transitioning to the Consolidated Rules for 2026, subject to specific overall applicability and effective dates within certain areas. |
| **FedRAMP Ready goes Legacy**                    | 2026-07-28 | No FedRAMP Ready submissions will be accepted after this date; providers should seek FedRAMP Rev5 Class A Certification instead.                                                              |
| FedRAMP Class A pipeline opens | 2026-08-03 | FedRAMP will begin accepting applications for FedRAMP 20x and Rev5 Class A Certifications. |
| Temporary Rev5 Program Certification Pipelines Open for Class B and C | 2026-08-10 | The Ready Conversion and Lost Sponsor pipelines open, allowing limited providers to apply for a Rev5 Class B or Class C Program Certification without an agency sponsor. See [Choosing a Certification Path](providers/start/path.md#temporary-rev5-program-certification-availability-for-class-b-and-c) for details. |
| FedRAMP 20x Class B & C pipeline opens | 2026-08-31 | FedRAMP will begin accepting applications for FedRAMP 20x Class B and C Certifications. |
| Mandatory Adoption                           | 2027-01-01 | The Consolidated Rules for 2026 take mandatory effect for all stakeholders, subject to specific overall applicability and effective dates within certain areas.                               |
| **End of New Rev5 Certifications** | 2027-06-11 | FedRAMP will no longer accept applications for new FedRAMP Rev5 Certifications. |
| Planned Release: Consolidated Rules for 2027 | 2027-07-01 | Yearly update to the Consolidated Rules targeted for release in mid-2027 and expiration in Dec 2029.                                                                                          |
| Planned Release: Consolidated Rules for 2028 | 2028-07-01 | Yearly update to the Consolidated Rules targeted for release in mid-2028 and expiration in Dec 2030.                                                                                          |
| Expiration of Consolidated Rules for 2026    | 2028-12-31 | The Consolidated Rules for 2026 reach the end of support and all stakeholders must complete their transition to either the Consolidated Rules for 2027 or 2028.                               |
